## Комментарии
Комментарии описывают что делает код и почему.

    // однострочный комментарий

    /*
        Многострочный
        комментарий
    */

JavaScript игнорирует код комментариев.

- код должен быть самодокументируемым
- используйте JSDoc-комментарии перед функциями
