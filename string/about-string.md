## UTF-16 characters, Unicode code points, and grapheme clusters
–°—Ç—Ä–æ–∫–∏ –ø—Ä–µ–¥—Å—Ç–∞–≤–ª—è—é—Ç—Å—è –≤ –æ—Å–Ω–æ–≤–Ω–æ–º –∫–∞–∫ –ø–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ—Å—Ç–∏ –∫–æ–¥–æ–≤—ã—Ö –µ–¥–∏–Ω–∏—Ü UTF-16. –í –∫–æ–¥–∏—Ä–æ–≤–∫–µ UTF-16 –∫–∞–∂–¥–∞—è –µ–¥–∏–Ω–∏—Ü–∞ –∫–æ–¥–∞ –∏–º–µ–µ—Ç –¥–ª–∏–Ω—É —Ä–æ–≤–Ω–æ 16 –±–∏—Ç. –≠—Ç–æ –æ–∑–Ω–∞—á–∞–µ—Ç, —á—Ç–æ —Å—É—â–µ—Å—Ç–≤—É–µ—Ç –º–∞–∫—Å–∏–º—É–º 216 –∏–ª–∏ 65536 –≤–æ–∑–º–æ–∂–Ω—ã—Ö —Å–∏–º–≤–æ–ª–æ–≤, –∫–æ—Ç–æ—Ä—ã–µ –º–æ–≥—É—Ç –±—ã—Ç—å –ø—Ä–µ–¥—Å—Ç–∞–≤–ª–µ–Ω—ã –∫–∞–∫ –æ—Ç–¥–µ–ª—å–Ω—ã–µ –∫–æ–¥–æ–≤—ã–µ –µ–¥–∏–Ω–∏—Ü—ã UTF-16. –≠—Ç–æ—Ç –Ω–∞–±–æ—Ä —Å–∏–º–≤–æ–ª–æ–≤ –Ω–∞–∑—ã–≤–∞–µ—Ç—Å—è –±–∞–∑–æ–≤–æ–π –º–Ω–æ–≥–æ—è–∑—ã—á–Ω–æ–π –ø–ª–æ—Å–∫–æ—Å—Ç—å—é (BMP) –∏ –≤–∫–ª—é—á–∞–µ—Ç –≤ —Å–µ–±—è –Ω–∞–∏–±–æ–ª–µ–µ —Ä–∞—Å–ø—Ä–æ—Å—Ç—Ä–∞–Ω–µ–Ω–Ω—ã–µ —Å–∏–º–≤–æ–ª—ã, —Ç–∞–∫–∏–µ –∫–∞–∫ –ª–∞—Ç–∏–Ω—Å–∫–∏–π, –≥—Ä–µ—á–µ—Å–∫–∏–π, –∫–∏—Ä–∏–ª–ª–∏—á–µ—Å–∫–∏–π –∞–ª—Ñ–∞–≤–∏—Ç—ã, –∞ —Ç–∞–∫–∂–µ –º–Ω–æ–≥–∏–µ –≤–æ—Å—Ç–æ—á–Ω–æ–∞–∑–∏–∞—Ç—Å–∫–∏–µ —Å–∏–º–≤–æ–ª—ã. –ö–∞–∂–¥–∞—è –µ–¥–∏–Ω–∏—Ü–∞ –∫–æ–¥–∞ –º–æ–∂–µ—Ç –±—ã—Ç—å –∑–∞–ø–∏—Å–∞–Ω–∞ –≤ –≤–∏–¥–µ —Å—Ç—Ä–æ–∫–∏ —Å —Å–∏–º–≤–æ–ª–æ–º `\u`, –∑–∞ –∫–æ—Ç–æ—Ä—ã–º —Å–ª–µ–¥—É—é—Ç —Ä–æ–≤–Ω–æ —á–µ—Ç—ã—Ä–µ —à–µ—Å—Ç–Ω–∞–¥—Ü–∞—Ç–µ—Ä–∏—á–Ω—ã—Ö —Ü–∏—Ñ—Ä—ã.

–û–¥–Ω–∞–∫–æ –≤–µ—Å—å –Ω–∞–±–æ—Ä —Å–∏–º–≤–æ–ª–æ–≤ Unicode –Ω–∞–º–Ω–æ–≥–æ –±–æ–ª—å—à–µ, —á–µ–º 65536. –î–æ–ø–æ–ª–Ω–∏—Ç–µ–ª—å–Ω—ã–µ —Å–∏–º–≤–æ–ª—ã —Ö—Ä–∞–Ω—è—Ç—Å—è –≤ UTF-16 –≤ –≤–∏–¥–µ —Å—É—Ä—Ä–æ–≥–∞—Ç–Ω—ã—Ö –ø–∞—Ä, –∫–æ—Ç–æ—Ä—ã–µ –ø—Ä–µ–¥—Å—Ç–∞–≤–ª—è—é—Ç —Å–æ–±–æ–π –ø–∞—Ä—ã 16-–±–∏—Ç–Ω—ã—Ö –∫–æ–¥–æ–≤—ã—Ö –µ–¥–∏–Ω–∏—Ü, –ø—Ä–µ–¥—Å—Ç–∞–≤–ª—è—é—â–∏—Ö –æ–¥–∏–Ω —Å–∏–º–≤–æ–ª. –í–æ –∏–∑–±–µ–∂–∞–Ω–∏–µ –¥–≤—É—Å–º—ã—Å–ª–µ–Ω–Ω–æ—Å—Ç–∏ –¥–≤–µ —á–∞—Å—Ç–∏ –ø–∞—Ä—ã –¥–æ–ª–∂–Ω—ã –Ω–∞—Ö–æ–¥–∏—Ç—å—Å—è –≤ –¥–∏–∞–ø–∞–∑–æ–Ω–µ –æ—Ç 0xD800 –¥–æ 0xDFFF, –∏ —ç—Ç–∏ –∫–æ–¥–æ–≤—ã–µ –µ–¥–∏–Ω–∏—Ü—ã –Ω–µ –∏—Å–ø–æ–ª—å–∑—É—é—Ç—Å—è –¥–ª—è –∫–æ–¥–∏—Ä–æ–≤–∞–Ω–∏—è —Å–∏–º–≤–æ–ª–æ–≤ —Å –æ–¥–Ω–æ–π –∫–æ–¥–æ–≤–æ–π –µ–¥–∏–Ω–∏—Ü–µ–π. (–¢–æ—á–Ω–µ–µ, —Å—Ç–∞—Ä—à–∏–µ —Å—É—Ä—Ä–æ–≥–∞—Ç—ã –∏–º–µ—é—Ç –∑–Ω–∞—á–µ–Ω–∏—è –æ—Ç 0xD800 –¥–æ 0xDBFF –≤–∫–ª—é—á–∏—Ç–µ–ª—å–Ω–æ, –∞ –º–ª–∞–¥—à–∏–µ —Å—É—Ä—Ä–æ–≥–∞—Ç—ã –∏–º–µ—é—Ç –∑–Ω–∞—á–µ–Ω–∏—è –æ—Ç 0xDC00 –¥–æ 0xDFFF –≤–∫–ª—é—á–∏—Ç–µ–ª—å–Ω–æ.) –ö–∞–∂–¥—ã–π —Å–∏–º–≤–æ–ª Unicode, —Å–æ—Å—Ç–æ—è—â–∏–π –∏–∑ –æ–¥–Ω–æ–π –∏–ª–∏ –¥–≤—É—Ö –∫–æ–¥–æ–≤—ã—Ö –µ–¥–∏–Ω–∏—Ü UTF-16, —Ç–∞–∫–∂–µ –Ω–∞–∑—ã–≤–∞–µ—Ç—Å—è –∫–æ–¥–æ–≤–æ–π —Ç–æ—á–∫–æ–π Unicode. –ö–∞–∂–¥–∞—è –∫–æ–¥–æ–≤–∞—è —Ç–æ—á–∫–∞ Unicode –º–æ–∂–µ—Ç –±—ã—Ç—å –∑–∞–ø–∏—Å–∞–Ω–∞ –≤ –≤–∏–¥–µ —Å—Ç—Ä–æ–∫–∏ —Å \u{xxxxxx}, –≥–¥–µ xxxxxx –ø—Ä–µ–¥—Å—Ç–∞–≤–ª—è–µ—Ç 1-6 —à–µ—Å—Ç–Ω–∞–¥—Ü–∞—Ç–µ—Ä–∏—á–Ω—ã—Ö —Ü–∏—Ñ—Ä.

"–û–¥–∏–Ω–æ–∫–∏–π —Å—É—Ä—Ä–æ–≥–∞—Ç" - —ç—Ç–æ 16-–±–∏—Ç–Ω–∞—è –∫–æ–¥–æ–≤–∞—è –µ–¥–∏–Ω–∏—Ü–∞, —É–¥–æ–≤–ª–µ—Ç–≤–æ—Ä—è—é—â–∞—è –æ–¥–Ω–æ–º—É –∏–∑ –ø—Ä–∏–≤–µ–¥–µ–Ω–Ω—ã—Ö –Ω–∏–∂–µ –æ–ø–∏—Å–∞–Ω–∏–π:
- –û–Ω –Ω–∞—Ö–æ–¥–∏—Ç—Å—è –≤ –¥–∏–∞–ø–∞–∑–æ–Ω–µ –æ—Ç 0xD800 –¥–æ 0xDBFF –≤–∫–ª—é—á–∏—Ç–µ–ª—å–Ω–æ (—Ç.–µ. —è–≤–ª—è–µ—Ç—Å—è —Å—Ç–∞—Ä—à–∏–º —Å—É—Ä—Ä–æ–≥–∞—Ç–æ–º), –Ω–æ —ç—Ç–æ –ø–æ—Å–ª–µ–¥–Ω—è—è –µ–¥–∏–Ω–∏—Ü–∞ –∫–æ–¥–∞ –≤ —Å—Ç—Ä–æ–∫–µ –∏–ª–∏ —Å–ª–µ–¥—É—é—â–∞—è –µ–¥–∏–Ω–∏—Ü–∞ –∫–æ–¥–∞ –Ω–µ —è–≤–ª—è–µ—Ç—Å—è –º–ª–∞–¥—à–∏–º –∑–∞–º–µ—Å—Ç–∏—Ç–µ–ª–µ–º.
