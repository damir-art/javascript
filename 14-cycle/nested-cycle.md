# Вложенные циклы
Вложенные циклы - это циклы вложенные друг в друга.

    let a = ''
    btn.addEventListener('click', function () {

        for (let i = 0; i < 5; i++) {
            for (let j = 0; j < 3; j++) {
                a += j + ' '
            }
            console.log(`${i}: ${a}`)
        }
        a = ''

    });

Вместо `j` лучше использовать `k`, чтобы больше визуально отличалось от `i`.
