    - внедрение
    - консоль console.log()

- Переменные (let, const)
- Работаем с элементами формы
- Работаем со стилями
- Типы данных (typeof)
- Операторы и операции
- Управляющие структуры
    - ветвления
    - циклы
- Массивы
- Асинхронность
- Сетевая работа
- BOM
- DOM
- События
- Нативные объекты
- Объекы
- ООП
