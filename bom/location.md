# location
Свойства объекта location.  
location - глоабльный объект, содержит информацию о текущем содержимом адресной строки.

Пример:

    hash - "",
    host - "127.0.0.1:5500",
    hostname - "127.0.0.1",
    href - "http://127.0.0.1:5500/test/index.html",
    ​origin - "http://127.0.0.1:5500",
    ​pathname - "/test/index.html",
    ​port - "5500",
    ​protocol - "http:"
​
